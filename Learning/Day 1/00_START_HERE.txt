What to practice:
  Go to your practice directory:
  $ cd ~/shell_practice
  
  Try these commands:
  
  $ ls -l readonly.txt           # See permissions: -r--r--r--
  $ cat readonly.txt             # Can read
  $ echo "test" > readonly.txt   # Can't write (permission denied!)
  
  $ chmod 644 readonly.txt       # Make it writable
  $ ls -l readonly.txt           # See new permissions: -rw-r--r--
  $ echo "test" > readonly.txt   # Now it works!
  
  $ chmod 444 readonly.txt       # Back to read-only
  $ chmod u+w readonly.txt       # Add write for user
  $ chmod go-r readonly.txt      # Remove read for group & others
  $ ls -l readonly.txt           # See the changes
  
  $ ls -l hello.sh               # See executable permission
  $ chmod -x hello.sh            # Remove execute
  $ ./hello.sh                   # Won't work now!
  $ chmod +x hello.sh            # Add execute back
  $ ./hello.sh                   # Works again!

Practice Exercise:
  Create your own file and practice all permission combinations:
  $ touch myfile.txt
  $ chmod 777 myfile.txt         # All permissions
  $ chmod 000 myfile.txt         # No permissions
  $ chmod 644 myfile.txt         # Standard file permissions
  $ chmod 755 myfile.txt         # Executable permissions
  $ ls -l myfile.txt             # Check after each change
